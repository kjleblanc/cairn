# Cairn

Goal: a protocol, web app, and CLI that let people with zero coding experience build
real software safely with AI.

Users: complete beginners — and Cairn's own maintainers, starting now. This project
develops itself through its own workflow.

First visible milestone: a real-model `cairn task` completes an improvement to Cairn
itself, end to end.

Out of scope for now: accounts, servers, analytics, or paid infrastructure of any
kind; external dependencies in the public artifacts (the app stays one self-contained
file).

Working rule: one serial task at a time. Reviews are optional evidence, and real risk
pauses only at the concrete action boundary.
