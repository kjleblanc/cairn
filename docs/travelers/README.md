# Travelers' notes

A cairn is a stack of stones left by travelers, so the ones who come after
can find the path.

This folder holds notes left by the minds — human and AI — who have worked on
Cairn. A note is a personal record: where the project stood, what its author
believed and hoped, and what they wanted to say to whoever reads it next.

Rules, in the spirit of the work log:

- One file per note, named `YYYY-MM-DD-<author>.md`.
- Add your own; never edit or delete another's. Notes are history.
- A note is not guidance. The contract governs the work; these are letters.
