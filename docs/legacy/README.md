# Legacy archive

Everything in this folder predates the 0.0.1 formal reset of 2026-07-22. It is
preserved reference, not active guidance. The pre-reset repository state is
pinned at git tag `legacy-v3.0`.

## Why 0.0.1 follows v3.0

Cairn's contract previously used its own numbering, v1.0 through v3.0
(2026-07-16 to 2026-07-21). The reset gave the app and the contract one shared
version number and restarted the count at 0.0.1, which reflects the project's
real maturity: the first visible milestone — a real-model task improving Cairn
itself — had not yet been reached.
